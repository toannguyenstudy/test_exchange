## EXCHANGE_TOANNGUYEN
### Using this project

 - Clone this project to computer
 - Use `npm install` to download all used package
 - Use command line `ionic serve` to run this project
 - For performance tesing, use Chrome mobile devtools (IOS or Android platform)
